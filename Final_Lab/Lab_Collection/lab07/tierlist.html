Reflection

Creating this Pokémon tier list helped get some practice using different HTML and CSS skills. By structuring the table using tr, td, and th tags, the table is neatly organized by Pokémon rankings into clear rows and columns. Incorporating images in the table taught us how to use the img helped reinforce something we already learned while giving more aesthetic value. Also, I used alt attributes to make sure that it is accessible. Lastly, I adjusted the width of the columns and rows to accommodate my data.
